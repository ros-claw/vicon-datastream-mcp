# Vicon DataStream MCP Server 示例提示词

## 基本连接
请帮我连接到本地的 Vicon 系统。

## 获取数据
获取 Colin 的骨盆位置和旋转数据。
列出所有可用的主体。

## 坐标系设置
设置成 Unity 坐标系（Y-up）。

## 设备数据
获取所有力板的数据。
获取眼动仪的注视向量。

## 完整流程示例
```
1. 连接到 Vicon 系统
2. 启用段数据和标记点数据
3. 获取 Colin 的骨盆姿态
4. 获取所有标记点位置
5. 获取力板数据
6. 分析系统延迟
7. 断开连接
```

## 常用操作
- 连接到 localhost:801
- 设置流模式为 ServerPush（低延迟）
- 只接收指定主体的数据
- 获取时间码信息
